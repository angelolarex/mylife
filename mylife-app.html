💾 Salvataggio & Sincronizzazione
ℹ️
Come funziona il salvataggioI dati vengono salvati automaticamente nel browser (localStorage). Sono legati a questo browser su questo PC.
📥 Per usarla sul cellulare o fare un backup:
1. Clicca Esporta Dati → salva il file mylife-backup.json
2. Manda il file sul telefono (WhatsApp, email, Drive…)
3. Apri l'app HTML sul telefono nel browser
4. Clicca Importa Dati e seleziona il file JSON

💡 Consiglio: salva l'HTML sul telefono o aprilo da Drive/iCloud per averlo sempre disponibile. Tieni il file HTML e il JSON backup nella stessa cartella.
